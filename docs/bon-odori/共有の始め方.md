# 参加可否を「保存したら全員に見える」ようにする（坂倉さん・5分）

## みんなの使い方（設定後）

1. [公開ページ](https://del10ro17-debug.github.io/bon-odori-roadmap/bon-odori/) を開く
2. **参加・回答** タブ
3. 名前・時間帯を選んで **回答を保存**  
→ 全員の画面に反映（30秒〜1分）

JSONやトークンは**不要**です。

---

## 坂倉さんが最初に1回だけやること

### ① Google Apps Script を開く

https://script.google.com/ → **新しいプロジェクト**

### ② コードを貼る

`コード.gs` の中身を**すべて削除**し、  
リポジトリの `docs/bon-odori/attendance-api.gs` を**全部コピーして貼り付け** → **保存**

### ③ setup を実行

1. 上の関数一覧で **`setup`** を選ぶ  
2. **実行** → 権限を許可  
3. 実行ログに `SPREADSHEET_ID` が出ればOK  
4. 続けて **`seedCurrentResponses`** を実行（いまの回答を入れる）

### ④ ウェブアプリとして公開

1. **デプロイ** → **新しいデプロイ**  
2. 種類: **ウェブアプリ**  
3. 実行ユーザー: **自分** / アクセス: **全員**  
4. **デプロイ** → 出た **URL**（`.../macros/s/.../exec`）をコピー

### ⑤ URL をページに設定

`docs/bon-odori/data.js` の `attendanceApiUrl` に貼る:

```javascript
attendanceApiUrl: "ここにコピーしたURL",
```

GitHub に push → 1〜2分後、全員が「保存だけ」で共有できます。

---

設定が終わると、ページ上部に **「自動共有: オン」** と表示されます。
