# 湾岸マンション価格DB データソース方針

このDBは、売出価格の観測値を自社分析に使うためのものです。成約価格ではないため、評価や売買判断では必ず「売出価格」と明記します。

## 優先するデータソース

1. Gmailで受信した価格アラート、市場レポート、仲介会社からの案内
2. 手動で確認した公開掲載情報の控え
3. 利用規約で許可されたAPI、CSV、データサービス

## SUUMOなど掲載サイトの扱い

- 自動巡回スクレイピングを初期運用の前提にしない。
- 取り込む前に、利用規約、robots.txt、アクセス負荷、再利用範囲を確認する。
- 許可範囲が不明な場合は、手動確認またはメールアラート経由に留める。
- 取得データを第三者へ再配布しない。

## 保存するメタデータ

- 観測日
- 出典種別
- 出典名
- 出典URLまたはメールID
- 売出価格、面積、間取り、階数、方角、棟
- 抽出元テキスト
- 抽出信頼度

## 品質管理

- HARUMI FLAGは `project_name`, `village_name`, `building_code` に正規化する。
- 同一掲載の重複、価格改定前後、掲載終了を分けて扱う。
- 外れ値は `source_text` と `raw_line` を確認してから分析に採用する。
- 成約価格、査定価格、売出価格を混在させない。
